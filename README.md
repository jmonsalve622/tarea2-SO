# Tarea 2 Sistemas Operativos

## Parte I. Sincronización con Barrera reutilizable

### Complilación
Para compilar el programa, ejecuta el siguiente comando en tu terminal:
```bash
gcc main.c -o main -pthread
```

### Ejecución
El programa se ejecuta con el siguiente formato:
```bash
./main [num_hebras] [num_etapas]
```

#### Argumentos:
- `[num_hebras]`: Números de hebras que se van a utilizar. Si no se declara, su valor será 5.
- `[num_etapas]`: Número de etapas en las que las hebras trabajarán. Si no se declara, su valor será 4.
